# SocialWeb_Express
website for social activity
# Getting started
1. put database.sql in your mysql
```
mysql db_name < database.sql
```
2. put your mysql db_user information by this command
```
npm run-script firstSet

host: localhost
user: root
password: password
```
3. install modules
```
npm install
```
4. start the server
```
npm start
```
